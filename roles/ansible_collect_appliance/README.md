# README

## Synopsis

This role is a technical unit of Unified Data Collector, not a standalone Ansible role.

The role provides data collection tasks for devices which can only be reached either via `ansible.builtin.raw` module or
by using `ansible_network_os` and `connection: ansible.netcommon.network_cli`
